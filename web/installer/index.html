What is MicriOS?

MicriOS is a tiny ESP32 app launcher with board-specific interfaces and shared app logic. The C3 build is designed around extreme constraints, the T-Display build uses the larger color screen for dashboards, and headless builds turn spare ESP32 boards into utility or miner nodes.

Before flashing

Use a data-capable USB cable, close any serial monitor using the port, and hold BOOT while connecting the board if it will not enter flashing mode.
